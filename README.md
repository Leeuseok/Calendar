### 💡 주요 기능
- **OAuth 2.0, Firebase Authentication**
  - 구글 이메일을 통한 회원가입
  
- **NestJS로 일정 데이터 관리**
  - 서버와 통신
  - 일정 CRUD 작업
 
- **Provider 상태 관리**
  - 일정 상태 관리
  - UI 업데이트 알림
  
- **AdMob**
  - 배너에 구글 애드몹을 활용해 광고 표시
